# Nova Stage Suite – Brand Guide

## Designprinzip
Die Website orientiert sich direkt an der Software-Oberfläche:
dunkler Hintergrund, technische Panels, Orange/Cyan-Akzente und klare Modul-Karten.

## Farben

| Name | Hex | Verwendung |
|---|---|---|
| Nova Background | #03070A | Seitenhintergrund |
| Nova Panel | #071116 | Cards, Panels, Modulflächen |
| Nova Orange | #FF6B00 | Hauptakzent, Buttons, Highlights |
| Nova Cyan | #00D9FF | Sekundärakzent, Linien, technische Hinweise |
| Nova Text | #F7F9FC | Haupttext |
| Nova Muted | #9CB0C2 | Beschreibungstext |

## Typografie
Empfohlen:
- Überschriften: Inter, Segoe UI, Roboto, Arial
- Fließtext: Inter, Segoe UI, Roboto, Arial

Die Website nutzt aktuell System-Fonts, damit keine externen Abhängigkeiten nötig sind.

## UI-Komponenten
- `.btn`
- `.btn-primary`
- `.btn-secondary`
- `.module-card`
- `.download-card`
- `.neon-panel`
- `.feature-split`
- `.check-list`

## Inhaltsregel
Es werden nur Funktionen gezeigt, die Nova Stage Suite aktuell tatsächlich besitzt.
Keine erfundenen Features, keine übertriebenen Versprechen.
