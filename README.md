# Nova Stage Suite Website v4

Saubere Struktur wie eine echte Software-Website.

## Struktur
- `index.html` – Startseite
- `download.html` – Downloadbereich
- `docs.html` – Dokumentation
- `faq.html` – FAQ
- `changelog.html` – Release Notes
- `modules/` – eigene Seiten für Launcher, Audio Renamer, Performance Monitor, Playlist Builder und Controller
- `assets/logo/` – Logo
- `assets/screenshots/` – Screenshots
- `css/main.css` – Designsystem
- `js/app.js` – Animationen und Mobile-Menü

## Wichtig
Alle Texte basieren auf den Funktionen, die du angegeben hast. Es wurden keine zusätzlichen Softwarefunktionen erfunden.

## Download-Links anpassen
In `download.html` nach `DEIN-NAME/DEIN-REPO` suchen und ersetzen.

## Hosting
